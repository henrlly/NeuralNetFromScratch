# NeuralNetFromScratch
 Neural net implemented from scratch using only Numpy

## Features
 - Linear layer
 - ReLu
 - Tanh
 - Sigmoid
 - Softmax
 - Dropout
 - Ensemble models
 - Save and load models

## To use
Run `run.ipynb`

Alternatively, `import nn` in your `.py`/`.ipynb` file and call classes/functions from there
